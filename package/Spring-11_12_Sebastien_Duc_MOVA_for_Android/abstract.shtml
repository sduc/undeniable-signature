Short, Undeniable Signature for Android 2.x
 
Author: Sebastien Duc
 
Supervisor(s): Dr. Ioana Boureanu
 
Date: Spring Semester 2012
 
Abstract:

In this project, we first studied MOVA signature scheme, a short, undeniable signature scheme. We tried to find an application for Android where MOVA can bring something more that classical signature schemes. The selected application is a University Contest on mobile phones. In this application, students in universities are forming teams and answer to quizzes made by the other universities. We built a threat model in this context and justified the use of MOVA. The signature are used mostly to avoid cheating. The two interesting properties of MOVA we exploited are shortess of the signature and the possibility to do a batch verification. The first is very convenient in a mobile environment. The second phase of this project consisted in implementing the application on Android and Java. We implemented both part of the application, which are the university servers and the android clients. We also implemented MOVA signature scheme.
